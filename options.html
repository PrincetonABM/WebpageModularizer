About

Modulr divides any webpage into various modules, which can be customized in a variety of ways (font size, color, type, deletion, isolation...). Customizations can be applied to individual modules or globally to the entire webpage. Configurations can be saved and reloaded for certain webpages and Modulr can be set to run on prescribed domains.

FAQ

Why can't I split certain modules?

Some modules that contain a short paragraph of text cannot be meaningfully divided (do you really want individual words being modules?). Other modules contain indivisible content, such as a video or an image. Moreover, some webpages with an unnatural webpage structure can have modules that just cannot be divided effectively.

What does merging a module do?

When a module is merged, the module is combined with other similar, nearby modules. For example, merging a paragraph within a news article will combine this paragraph with every other paragraph in the article.

Some webpages don't modularize that well.

Webpages are diverse, so it is nearly impossible to create an application that can work effectively on every webpage. Thankfully, that is why split and merge functions are included for every module. If a module is too big, split it. If a module is too small, merge it.
